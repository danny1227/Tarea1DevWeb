Curso de Base de Datos 1

Descripción del curso

El curso de Base de Datos 1 introduce los fundamentos de los sistemas de gestión de bases de datos, incluyendo diseño, implementación y manejo de bases de datos relacionales.

Objetivos del curso

  • Comprender los conceptos básicos de bases de datos y sistemas de gestión de bases de datos.
  • Diseñar bases de datos utilizando modelos entidad-relación.
  • Implementar bases de datos relacionales.
  • Realizar consultas y manipulación de datos usando SQL.
  • Gestionar la integridad y seguridad de los datos.

Contenido del curso

  1. Introducción a las bases de datos
  2. Modelado entidad-relación
  3. Diseño de bases de datos relacionales
  4. Lenguaje SQL
  5. Normalización de bases de datos
  6. Gestión de transacciones y concurrencia

Datos interesantes sobre la materia

Dato Descripción
Historia Las bases de datos relacionales fueron propuestas por primera vez por E.F. Codd en 1970.
Aplicaciones Las bases de datos son esenciales en sistemas como los de banca, reservas aéreas, y gestión de inventarios.
SQL El lenguaje SQL (Structured Query Language) es el estándar para la gestión de bases de datos relacionales.
